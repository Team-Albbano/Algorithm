풀이 설명란

- (1) 처음 정답을 넣을 수 있는 배열을 생성해줍니다.
- (2) for문을 돌려 n까지 반복하여 나머지가 0 이 나오는 숫자들을 정답 배열에 푸쉬해줍니다.
- (3) 나온 배열들의 length를 return 해주면 result 값이 나옵니다.

```js
function solution(n) {
  const result = [];
  for (let i = 1; i <= n; i++) {
    if (n % i === 0) result.push(i);
  }
  return result.length;
}
```
