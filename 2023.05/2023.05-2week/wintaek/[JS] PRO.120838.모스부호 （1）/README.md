## 풀이

풀이 설명란

- (1) morse 라는 객체를 하나 만들어줍니다.
- (2) 배열로 돌릴 수 있게끔 split 을 사용합니다.
- (3) map 으로 전체적으로 돌린 후 index를 찾아야하니 [a] 를 추가해줍니다.
- (4) 배열을 다시 문자열로 돌릴 수 있게 join 을 사용합니다.

```javascript
function solution(letter) {
  const morse = {
    ".-": "a",
    "-...": "b",
    "-.-.": "c",
    "-..": "d",
    ".": "e",
    "..-.": "f",
    "--.": "g",
    "....": "h",
    "..": "i",
    ".---": "j",
    "-.-": "k",
    ".-..": "l",
    "--": "m",
    "-.": "n",
    "---": "o",
    ".--.": "p",
    "--.-": "q",
    ".-.": "r",
    "...": "s",
    "-": "t",
    "..-": "u",
    "...-": "v",
    ".--": "w",
    "-..-": "x",
    "-.--": "y",
    "--..": "z",
  };
  return letter
    .split(" ")
    .map((a) => morse[a])
    .join("");
}
```
